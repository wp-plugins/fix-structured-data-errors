This plugin will add structured data tags to your pages and posts in an attempt to fix common structured data errors.

== Description ==

This plugin is only available here: https://www.acceleratormarketing.com/product/fix-structured-data-errors-wordpress-plugin/

== Installation ==

This section describes how to install the plugin and get it working.

e.g.

1. Upload `fix-structured-data-errors.php` to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress

== Frequently Asked Questions ==

= Do I need to configure any settings? =

No, the plugin automatically adds structured data to your posts and pages which should fix errors in most cases.
